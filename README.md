# 💼 Personal Portfolio Website

A modern, responsive, and dynamic personal portfolio website built using **HTML, CSS, and JavaScript**. This portfolio showcases my skills, projects, education, achievements, and contact information through an interactive and visually appealing user interface.

## ✨ Features

- 🎨 Modern & Professional UI Design
- 📱 Fully Responsive for Mobile, Tablet, and Desktop
- 🌙 Dark/Light Theme Toggle
- ⌨️ Typing Text Animation
- 🚀 Smooth Scrolling Navigation
- 💻 Project Showcase Section
- 📚 Education Timeline
- 📊 Animated Skills Progress Bars
- 📩 Contact Form
- ⚡ Interactive Animations & Hover Effects
- 🎯 Clean and Maintainable Code Structure

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)

1. Clone this repository


2. Navigate to the project folder

